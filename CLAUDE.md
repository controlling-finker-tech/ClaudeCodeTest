## Project Overview

This is a single-file web project — a browser-based Tic Tac Toe game. There is no build system, package manager, or framework. The entire app lives in `tictactoe.html` (HTML + CSS + JS in one file).

To run it, open `tictactoe.html` in a browser (double-click or use `powershell.exe -Command "Invoke-Item 'tictactoe.html'"`).
